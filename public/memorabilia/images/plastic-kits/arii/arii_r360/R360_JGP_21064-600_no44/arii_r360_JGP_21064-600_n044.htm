1/32 Arii Mazda R360 JGP.

Owners Club Series #21064-600 NO.44
   
This kit was first issued by "LS" kit #2151 500 No.18. Arii has made some very minor changes to it for e.g. the Arii model come in a bigger box 22.5cm(L)x15.5cm(W)x4.5cm(H) compared with the "LS" box of 95cm(L)x13cm(W)x4cm(H). Arii has also made changes to the decals, which is a bit of a pity because now it is not as historically accurate as the "LS" kit was.

This kit is basically just Arii kit # 41015-600 #15 with some additional parts such as flares for the wheel arches option mufflers, different wheels and decals.
