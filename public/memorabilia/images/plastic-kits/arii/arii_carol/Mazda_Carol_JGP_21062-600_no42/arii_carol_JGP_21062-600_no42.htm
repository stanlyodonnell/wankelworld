1/32 Arii Mazda Carol JGP.

Owners Club Series #21062-600 NO.42
This kit was first issued by "LS" kit #. Arii has made some very minor changes to it for e.g. the Arii model come in a bigger box 22.5cm(L)x15.5cm(W)x4.5cm(H) compared with the "LS" box of 19.5cm(L)x13cm(W)x4cm(H). Arii has also made changes to the decals, which is a bit of a pity because now it is not as historically accurate as the "LS" kit was.

This kit is basically just Arii kit #410008-600 NO.8 with some additional parts such as flares for the wheel arches option mufflers, different wheels and decals.
