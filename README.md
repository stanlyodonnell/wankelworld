# Wankelworld

A modern website celebrating Mazda rotary engines and the legacy of Felix Wankel.

## About

Wankelworld is the spiritual successor to DrMaz.com, a legendary rotary enthusiast website from the early 2000s. This modern rebuild preserves the knowledge and passion of the original while bringing it into 2026 with responsive design, clean code, and fast performance.

## Features

- **Responsive Design** - Works beautifully on desktop, tablet, and mobile
- **Fast Loading** - Static HTML/CSS/JS with no dependencies
- **Dark Theme** - Easy on the eyes with Mazda-inspired Soul Red Crystal accents
- **Complete Content** - Comprehensive coverage of rotary history, models, and culture

## Site Structure

```
public/
├── index.html          # Homepage with hero and featured sections
├── mazda.html          # Complete Mazda rotary model history
├── rotary.html         # Rotary engine technical deep-dive
├── felix-wankel.html   # Biography of the inventor
├── aviation.html       # Rotary engines in aircraft
├── memorabilia.html    # Collectibles and model guide
├── gallery.html        # Photo gallery and media
├── links.html          # Resources and community links
├── css/
│   └── main.css        # Complete stylesheet
└── js/
    └── main.js         # Interactive functionality
```

## Getting Started

### Local Development

```bash
cd wankelworld
npx serve public
```

Or simply open `public/index.html` in your browser.

### Deployment

The `public/` folder contains everything needed to deploy. Upload to any static hosting:

- Netlify
- Vercel
- GitHub Pages
- Traditional web hosting

## Content Highlights

### Mazda Models
Complete coverage from the 1967 Cosmo Sport to the 2012 RX-8:
- Cosmo Sport 110S
- R100 / Familia Rotary
- RX-2, RX-3, RX-4, RX-5
- RX-7 (SA/FB, FC, FD generations)
- RX-8 and Renesis engine
- Concept cars

### Rotary History
Technical deep-dive into the Wankel engine:
- How it works
- Advantages and challenges
- Engine evolution (10A → 12A → 13B → 20B → Renesis)
- R26B Le Mans racing engine

### Felix Wankel
Biography of the German engineer who invented the rotary engine:
- Early life and inspiration
- The invention process
- Partnership with Mazda
- Lasting legacy

### Additional Sections
- **Aviation** - Rotary-powered aircraft and UAVs
- **Memorabilia** - Die-cast models, resin kits, collecting guide
- **Gallery** - Photo and video collections
- **Links** - Resources, forums, suppliers, clubs

## Design System

### Colors
- Primary: `#A6192E` (Mazda Soul Red Crystal)
- Dark Background: `#0D0D0D`
- Light Text: `#FFFFFF`
- Gray Scale: Various grays for hierarchy

### Typography
- Headings: Orbitron (tech/motorsport feel)
- Body: Inter (clean, readable)

### Logo
**WANKELWORLD** with "WANKEL" in white and "WORLD" in Mazda red
