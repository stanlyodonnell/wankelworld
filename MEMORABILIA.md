# Wankel World memorabilia section - DrMaz structure with modern dark theme

## Structure
- Main page: `/memorabilia.html` - 5 category cards
- Category pages:
  - `/memorabilia/plastic-kits.html` - Data table with manufacturer headers
  - `/memorabilia/resin-kits.html` - Data table with manufacturer headers  
  - `/memorabilia/tomica.html` - Data table with manufacturer headers
  - `/memorabilia/slot-cars.html` - Data table with manufacturer headers
  - `/memorabilia/diecast.html` - Data table with manufacturer headers
- Individual model pages: `/memorabilia/plastic-kits/[model-name].html`

## Table Columns
- Picture (thumbnail linking to detail page)
- Scale
- Manufacturer
- Item (name linking to detail page)
- Packaging
- Id#
- Country (flag emoji)

## Styling
- Dark theme (#0D0D0D background)
- Mazda Soul Red Crystal (#A6192E) for accents
- WANTED badges in red
- Orbitron headings
- Inter body text
