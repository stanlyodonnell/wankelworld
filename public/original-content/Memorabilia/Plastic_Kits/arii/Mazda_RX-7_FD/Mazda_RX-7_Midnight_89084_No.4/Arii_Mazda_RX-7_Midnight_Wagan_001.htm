1/32 Arii Midnight FD RX-7

Kit No. 86084 #4

This 1/32 scale Wangan Midnight (Japanese comic) kit # 86084-800 is the same kit as Arii kit #91064-600 it uses all the same parts which are all molded in the same colours as Arii kit #91064-600 . But one interesting note with this kit is, for some reason Arii decided not to make this model in Japan like they have with all their previous models, but instead this model is made in China. Well lets hope this is not the start of a new trend.
